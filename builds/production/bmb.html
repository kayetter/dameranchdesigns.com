Technical Design

Technical Summary Bodymindbusiness.com is a 6-page mobile responsive design modularized for reusable content using PHP. Secure https protocol, compressed images, asynchronous and minified javascript and css and cached files were used to improve performance and enhance security

Social Media Facebook and twitter card headers allow users to post bodymindbusiness.com links in a professionally designed format. LinkedIn, twitter, and Facebook sharing buttons are included on the 'Contact Us' page.

Dynamic Content Dynamic content including menu options and carousel testimonials was created using jquery-cycle, mustache and jquery UI. Users can email Deb using a prepopulated and editable email contactform.
